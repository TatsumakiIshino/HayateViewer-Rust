# HayateViewer Rust

Direct2D を描画バックエンドに使用した、高速・軽量な画像・自炊コミックビューアです。
Python版 HayateViewer のコンセプトを継承し、更なるパフォーマンスを追求して Rust で再構築されています。

![Settings Screenshot](file:///C:/Users/Tatsumaki/.gemini/antigravity/brain/ce66b745-8a9e-4822-8053-62b75d676c0e/uploaded_image_1766270453034.png)

## 主な特徴

- **圧倒的な描画速度**: Direct2D によるハードウェアアクセラレーションを活用。
- **2層キャッシュシステム**: CPU (メモリ) と GPU (VRAM) の両方で画像をキャッシュし、スムーズなページめくりを実現。
- **多様なフォーマットへの対応**:
  - 画像: JPEG, PNG, BMP, WEBP, **JPEG 2000 (JP2)**
  - 書庫: ZIP (CBZ), 7z (CB7)
- **快適な閲覧機能**:
  - 見開き表示（左右綴じ対応、表紙単独表示）
  - リアルタイムルーペ機能（右クリック）
  - スムーズなズーム・パン
- **設定 UI**: アプリ内オーバーレイによる日本語設定画面。

## システム要件

- OS: Windows 10 / 11 (64bit)
- グラフィックス: DirectX 11.0 以上に対応した GPU

## クイックスタート

### 開発用ビルドと実行

```powershell
cargo run
```

### リリース用ビルド

```powershell
cargo build --release
```

バイナリは `target/release/hayate-viewer-rust.exe` に生成されます。リリース版ではコンソールウィンドウが非表示になります。

## 操作方法

| キー / マウス | 動作 |
|:---|:---|
| `左右キー` / `ホイール` | ページ移動 |
| `O` (オー) | 設定画面を開く / 閉じる |
| `B` | 見開き・綴じ方向の切り替え |
| `右クリック` | ルーペ（拡大表示） |
| `左ドラッグ` | パン（移動） |
| `テンキー *` | 表示リセット |
| `Esc` | 設定画面を閉じる |
