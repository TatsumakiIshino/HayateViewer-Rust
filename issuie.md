# HayateViewer Rust

## 未解決の問題

- OpenGLのとき、`Shift + S` ページジャンプ UI の表示がおかしい。消すこともできない。
- `S` シークバーの表示中、バーの表示はされているが、マウス左ドラッグで動かそうとしても、バーが動かない。

## 未実装の機能

- レンダリングエンジンにWebGLを追加する。
- オプション画面の情報欄をもう少しかっこよく表示する。
- ルーペ機能の倍率をオプション画面で設定できるようにする。
- オプション画面で項目を選択したあと左クリック、またはスペースキーで内容を変更できるが、矢印キー左右でも項目変更できるようにしたい。
- オプション画面を表示中、例えば 'B' を押すと表示モードが変更されるが、オプション画面の該当項目にリアルタイムに反映させたい。
